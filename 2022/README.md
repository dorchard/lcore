# How to install

Requires GHC Haskell and the `stack` tool. Install [stack](https://docs.haskellstack.org/en/stable/) first, download the code,
then run:

```
stack install
```

Which should install the `lcore` program.

# Example usages

```
lcore programs/simple.lam
```
